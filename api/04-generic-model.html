    # get $key

    • $key ⇒ string

    Returns a key that can be used for v-for loops:

    <li v-for="task in tasklist.items" :key="$key">
  {{task.name}}
</li>

    # setValidationBehaviour

    • setValidationBehaviour( string ) ⇒ null

    Valid values are ERROR, WARNING, ELEMENT_VALIDATION, NONE
